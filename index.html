Design language

One gradient, four atmospheres.

Sora's color system is built from a single dusk-to-dawn wash — the same gradient that opens the app. Each mode pulls one hue from it and bathes the screen in that color, so the phone feels like the moment you're in. Action colors stay constant across modes (indigo to act, coral to escalate, mint to confirm), so muscle memory is never broken when the atmosphere changes.

Nightwalk

#D9DEFB · Periwinkle

The blue of streetlights on wet pavement. Cool, awake, vigilant without being anxious.

Dating

#F8C9C2 · Blush coral

Skin-warm and intimate, one shade away from the escalate-coral. Warmth and alertness live in the same family on purpose.

Commute

#C6E9DC · Mint

"Go" green softened to a routine. The most ordinary mode gets the most reassuring color.

Traveling

#E3D5F0 · Lilac

The in-between hour. Unfamiliar, jet-lagged, dreamlike — borrowed from the middle of the splash gradient.

The action layer

Three colors that mean the same thing in every mode — so users learn the buttons once and never re-learn them.

Indigo · Act Sora primary · #4A5DBE

The brand color. Walk me home. Start timer. Every primary button.

Coral · Escalate Alert accent · #E97B6B

Tier-3 emergency, watch alert bell, "Escalate" secondary CTA. Saved for one job.

Mint · Confirm Safety green · #3DA67A

"I'm safe" on the watch, Tier-1 silent notify, arrival check-in.

Stealth · The dark variant

When the screen can't be seen, the mode tint disappears entirely — only the action colors survive on near-black #0F1115. Indigo lifts to #8A9BE8 for AA contrast. Same flows, quieter face.
